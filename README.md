# Lista de Tarefas — Next.js 15

Projeto desenvolvido utilizando Next.js 15, TypeScript, App Router, Server Components, Client Components e testes unitários com Jest e Testing Library.

---

# Funcionalidades

- Listagem de tarefas
- Adição de novas tarefas
- Hook customizado para contagem
- Dados simulados via Promise.resolve()
- Testes unitários

---

# Tecnologias

- Next.js 15
- React
- TypeScript
- Jest
- Testing Library

---

# Instalação

```bash
npm install
```

---

# Executar projeto

```bash
npm run dev
```

Acesse:

```txt
http://localhost:3000
```

---

# Executar testes

```bash
npm test
```
